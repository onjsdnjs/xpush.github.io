Xpush

new Xpush(host, appId, eventHandler, autoInitFlag)

Xpush의 생성자

Parameters:
Name Type Argument Description
host string

접속할 Session Server의 address
appId string

application id
eventHandler string <optional>

session event를 처리하기 위한 함수
autoInitFlag boolean <optional>

자동 초기화 여부에 대한 flag
Source:
  • xpush.js, line 20
Examples
// Create new Xpush Object
var xpush = new Xpush( 'http://stalk-front-s01.cloudapp.net:8000', 'sample' );
// Create new Xpush Object with event Handler
var xpush = new Xpush( 'http://stalk-front-s01.cloudapp.net:8000', 'sample', function (type, data){
  console.log( " type : ", type );
  console.log( " data : ", data );
});
